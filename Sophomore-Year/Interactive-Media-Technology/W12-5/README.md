# 文字构建师 - 组合文字构建意义

**课程**: [Sophomore Year] Interactive Media Technology (互动媒体技术)  
**作业序号**: W12-5  
**作业类型**: 分组创作游戏

## 游戏简介

"文字构建师"是一款基于大语言模型的创意文字游戏。玩家通过输入关键词，AI会根据这些词语组合生成一个完整、有意义的故事。游戏体现了"组合文字构建意义"的核心概念——通过不同词语的组合，创造出全新的意义和故事。

## 游戏特色

- **组合创造意义**：玩家输入多个关键词，AI智能组合这些词语，构建完整的故事
- **双API支持**：支持DeepSeek API和阿里百炼（DashScope）API
- **灵活配置**：游戏开始界面提供API Key输入，玩家可自行配置
- **纯页面版**：无需安装，直接在浏览器中运行
- **美观界面**：现代化的UI设计，流畅的用户体验

## 游戏玩法

1. **配置API**
   - 选择使用的API类型（DeepSeek 或 阿里百炼）
   - 输入您的API Key
   - 可选择自定义API URL（默认使用官方地址）

2. **输入关键词**
   - 在输入框中输入词语，用空格或逗号分隔
   - 按回车键或点击输入框外部添加词语
   - 可以添加多个词语，每个词语会显示为标签

3. **生成故事**
   - 点击"生成故事"按钮
   - AI会根据您输入的词语组合，创作一个300-500字的故事
   - 故事会自然地融合所有关键词

4. **保存与分享**
   - 可以保存生成的故事为文本文件
   - 可以重新生成，探索不同的故事可能性

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 在开始界面配置API信息
3. 点击"开始游戏"进入游戏界面
4. 输入关键词并生成故事

## API Key获取

### DeepSeek API
1. 访问 [DeepSeek官网](https://www.deepseek.com/)
2. 注册账号并获取API Key
3. 在游戏开始界面输入API Key

### 阿里百炼API
1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册账号并获取API Key
3. 在游戏开始界面输入API Key

## 注意事项

- 需要有效的API Key才能使用
- 确保网络连接正常，API调用需要访问外部服务
- 生成故事的质量取决于输入的关键词和API的响应
- 建议使用有意义的、相关的关键词组合，以获得更好的故事效果

## 游戏理念

"组合文字构建意义"体现了语言和创造力的本质：
- **组合性**：单个词语本身有意义，但组合后能产生新的意义
- **创造性**：通过不同的组合方式，可以创造出无限可能的故事
- **协作性**：玩家提供关键词，AI负责构建，共同完成创作
