# Homework Repository

本仓库用于存储不同课程的作业，按学年、课程和作业序号进行组织。

## 📁 目录结构

```
homework/
├── 大三作业/                    # 大三学年
│   ├── Interactive-Media-Technology/  # 互动媒体技术
│   │   └── W12-5/              # 第12周第5次作业
│   │       ├── index.html
│   │       ├── style.css
│   │       ├── script.js
│   │       └── README.md
│   └── 知识工程作业/             # 知识工程
│       ├── build_medicalgraph.py
│       ├── chatbot_graph.py
│       └── ...
└── README.md                    # 本文件
```

## 🗂️ 命名规范

- **学年目录**: `大{Grade}作业` (例如: `大三作业`) 或者 `{Year}-Year`
- **课程目录**: 使用课程英文名称，单词首字母大写，用连字符连接 (例如: `Interactive-Media-Technology`)
- **作业目录**: 使用作业编号 (例如: `W12-5` 表示第12周第5次作业)

## 📝 添加新作业

1. 在对应的课程目录下创建新的作业目录
2. 按照命名规范命名目录
3. 在作业目录中添加 `README.md` 说明作业内容
4. 更新本文件的课程列表
